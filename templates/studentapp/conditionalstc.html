Conditional Statements in C

Conditional statements allow decision-making based on specific conditions, controlling the programâ€™s flow.

They are essential for creating flexible and responsive code in C programming.

1. If Statement

The if statement tests a condition and executes a block of code only if that condition is true.

Syntax:

if (condition) {
   // Code to execute if condition is true
}

Example:

Sample Code

#include <stdio.h>
int main() {
   int num = 10;
   if (num > 5) {
       printf("The number is greater than 5\n");
   }
   printf("This line executes regardless of the condition.\n");
   return 0;
}

Output:

The number is greater than 5
This line executes regardless of the condition.

2. If-Else Statement

The if-else statement adds an alternative block of code if the condition is false,

allowing the program to execute one block for true conditions and another for false ones.

Syntax:

if (condition) {
   // Code for true condition
} else {
   // Code for false condition
}

Example:

Sample Code

#include <stdio.h>
int main() {
   int num = 3;
   if (num > 5) {
       printf("Number is greater than 5\n");
   } else {
       printf("Number is not greater than 5\n");
   }
   return 0;
}

Output:

Number is not greater than 5

3. Nested If-Else

A nested if-else is an if statement within another if or else statement, allowing for more complex decision-making scenarios.

Syntax:

Sample Code

if (condition1) {
   if (condition2) {
       // Code for condition1 and condition2 being true
   } else {
       // Code for condition1 true, condition2 false
   }
} else {
   // Code if condition1 is false
}

Example:

Sample Code

#include <stdio.h>
int main() {
   int num = 10;
   if (num > 5) {
       if (num < 15) {
           printf("Number is between 5 and 15\n");
       } else {
           printf("Number is greater than or equal to 15\n");
       }
   } else {
       printf("Number is less than or equal to 5\n");
   }
   return 0;
}

Output:

Number is between 5 and 15

If-Else If Ladder in C

The if-else if ladder in C programming is used to test a series of conditions sequentially. Each condition is checked only if all preceding conditions are false.

If any conditional expression evaluates to true, the corresponding code block is executed, and the entire ladder is terminated.

Syntax:

// any if-else ladder starts with an if statement only
if(condition) {
   // code block
}
else if(condition) { 
   // this else if will be executed when the previous if condition is false
}
... // multiple else if statements can follow
else { 
   // final else block
}

Working Flow:

  1. The flow starts with the first if statement.
  2. The first condition is tested.
  3. If true, the corresponding block executes. If false, the flow moves to the next condition.
  4. This process continues until a true condition is found or all conditions are evaluated.
  5. If none are true, the else block (if present) executes.
  6. Exits the if-else-if ladder.

Note:

An if-else ladder can exclude the else block.

Example 1: Check whether a number is positive, negative, or 0

Sample Code

#include <stdio.h>

int main() {
   int n = 0;

   // Check if the number is positive, negative, or zero
   if (n > 0) {
       printf("Positive");
   }
   else if (n < 0) {
       printf("Negative");
   }
   else {
       printf("Zero");
   }
   return 0;
}

Output:

Zero

Example 2: Calculate Grade According to Marks

Sample Code

#include <stdio.h>

int main() {
   int marks = 91;
   
   // Determine grade based on marks
   if (marks <= 100 && marks >= 90)
       printf("A+ Grade");
   else if (marks < 90 && marks >= 80)
       printf("A Grade");
   else if (marks < 80 && marks >= 70)
       printf("B Grade");
   else if (marks < 70 && marks >= 60)
       printf("C Grade");
   else if (marks < 60 && marks >= 50)
       printf("D Grade");
   else
       printf("F Failed");

   return 0;
}

Switch Case Statement in C

The switch case statement evaluates a given expression and executes statements based on its value.

is used to perform different actions based on different conditions (cases) and serves as a substitute for lengthy if statements that compare a variable to multiple integral values.

Syntax of Switch Statement in C:

Sample Code

switch(expression)
{
   case value1: statement_1;
                break;
   case value2: statement_2;
                break;
   ...
   case value_n: statement_n;
                 break;
   default: default_statement;
}

Rules of the Switch Case Statement:

  • The "case value" must be of char or int type.
  • There can be one or multiple cases.
  • Case values must be unique.
  • A break statement is optional but can be used to exit the switch statement.
  • The default statement is optional.

Example: C Program to Demonstrate Switch Case

Sample Code

#include <stdio.h>

int main() {
   int var = 1;

   switch (var) {
       case 1:
           printf("Case 1 is Matched.");
           break;

       case 2:
           printf("Case 2 is Matched.");
           break;

       case 3:
           printf("Case 3 is Matched.");
           break;

       default:
           printf("Default case is Matched.");
           break;
   }

   return 0;
}

Output:

Case 1 is Matched.

How the Switch Statement Works:

  1. The switch variable is evaluated.
  2. The evaluated value is matched against all present cases.
  3. If a matching case value is found, the associated code is executed.
  4. If no match is found, the default case (if present) is executed.
  5. If a break keyword is present, control exits the switch statement. If omitted, execution continues to the next case.
  6. Statements after the switch statement are executed.

Example of Switch Case Without Break:

Sample Code

#include <stdio.h>

int main() {
   int var = 2;

   switch (var) {
       case 1:
           printf("Case 1 is executed.\n");
       case 2:
           printf("Case 2 is executed.\n");
       case 3:
           printf("Case 3 is executed.");
       case 4:
           printf("Case 4 is executed.");
   }
   return 0;
}

Output:

Case 2 is executed.
Case 3 is executed.
Case 4 is executed.

Default in Switch Case:

The default keyword specifies the set of statements to execute if there is no case match. It is optional and can be placed anywhere in the switch block.

Important Points About Switch Case Statements:

  1. The switch expression must result in a constant value.
  2. The expression value must be of int or char type.
  3. Case values must be unique.
  4. Nesting of switch statements is allowed but should be avoided for readability.
  5. The default block can be placed anywhere.

Example 1: Print the Day of the Week

Sample Code

#include <stdio.h>

// Driver Code
int main() {
   int day = 2;

   printf("The day with number %d is ", day);
   switch (day) {
       case 1:
           printf("Monday");
           break;
       case 2:
           printf("Tuesday");
           break;
       case 3:
           printf("Wednesday");
           break;
       case 4:
           printf("Thursday");
           break;
       case 5:
           printf("Friday");
           break;
       case 6:
           printf("Saturday");
           break;
       case 7:
           printf("Sunday");
           break;
       default:
           printf("Invalid Input");
           break;
   }
   return 0;
}

Output:

The day with number 2 is Tuesday

Example 2: Simple Calculator Using Switch Case

Sample Code

#include <stdio.h>
#include <stdlib.h>

// Driver code
int main() {
   char choice;
   int x, y;

   while (1) {
       printf("Enter the Operator (+,-,*,/)\nEnter x to exit\n");
       scanf(" %c", &choice);

       if (choice == 'x') {
           exit(0);
       }

       printf("Enter the two numbers: ");
       scanf("%d %d", &x, &y);

       switch (choice) {
           case '+':
               printf("%d + %d = %d\n", x, y, x + y);
               break;
           case '-':
               printf("%d - %d = %d\n", x, y, x - y);
               break;
           case '*':
               printf("%d * %d = %d\n", x, y, x * y);
               break;
           case '/':
               printf("%d / %d = %d\n", x, y, x / y);
               break;
           default:
               printf("Invalid Operator Input\n");
       }
   }
   return 0;
}

Output:

Sample Code

Enter the operator (+, -, *, /)
Enter x to exit
+
Enter the two numbers: 100 200
100 + 200 = 300

Advantages of C Switch Statement:

  • Easier to read than multiple if-else statements.
  • Faster execution speed.
  • Easier to debug and maintain with a large number of conditions.

Disadvantages of C Switch Statement:

  • Can only evaluate int or char types.
  • No support for logical expressions.
  • Must include a break statement in every case to prevent fall-through.
