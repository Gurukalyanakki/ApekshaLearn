Basic Input and Output in C

scanf()

The scanf() method in C reads a value from the console as per the specified type and stores it in the provided address.

Syntax: scanf("%X", &variableOfXType);

printf()

The printf() method in C prints the specified value to the console.

Syntax: printf("%X", variableOfXType);

Examples of Input and Output for Basic Types

Sample Code


#include <stdio.h>

int main()
{
   // Declare the variables
   int num;
   char ch;
   float f;

   // --- Integer ---
   printf("Enter the integer: ");
   scanf("%d", &num);
   printf("\nEntered integer is: %d", num);

   // --- Float ---
   while((getchar()) != '\n'); // Clearing buffer
   printf("\n\nEnter the float: ");
   scanf("%f", &f);
   printf("\nEntered float is: %f", f);

   // --- Character ---
   printf("\n\nEnter the Character: ");
   scanf("%c", &ch);
   printf("\nEntered character is: %c", ch);

   return 0;
}

Output

Enter the integer: 10
Entered integer is: 10

Enter the float: 2.5
Entered float is: 2.500000

Enter the Character: A
Entered Character is: A

Examples of Input and Output for Advanced Types

Sample Code


#include <stdio.h>

int main()
{
// Declare string variable
char str[50];

// Input the Word
printf("Enter the Word: ");
scanf("%s", str);
printf("\nEntered Word is: %s", str);

// Input the Sentence
printf("\n\nEnter the Sentence: ");

printf("\nEntered Sentence is: %s", str);

return 0;
}

Output

Enter the Word: APEKSHA
Entered Word is: APEKSHA

Enter the Sentence: APEKSHA LEARN
Entered Sentence is: APEKSHA LEARN

Relational Operators in C

Relational operators are used to compare two values. They return 1 (true) if the comparison is correct and 0 (false) otherwise.

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to

Example 1: Using == and !=

#include <stdio.h>

int main() {
    int a = 5, b = 10;

    // Check if a is equal to b
    if (a == b) {
        printf("a is equal to b\\n");
    } else {
        printf("a is not equal to b\\n");
    }

    // Check if a is not equal to b
    if (a != b) {
        printf("a is not equal to b\\n");
    }

    return 0;
}

Example 2: Using > and <=

#include <stdio.h>

int main() {
    int x = 15, y = 20;

    if (x > y) {
        printf("x is greater than y\\n");
    } else {
        printf("x is not greater than y\\n");
    }

    if (y <= 20) {
        printf("y is less than or equal to 20\\n");
    }

    return 0;
}

List of Format Specifiers in C

The format specifier in C is used to tell the compiler about the type of data to be printed or scanned in input and output operations.

They always start with a % symbol and are used in the formatted string in functions like printf(), scanf(), sprintf(), etc.

The C language provides a number of format specifiers that are associated with different data types,

such as %d for int, %c for char, and more. Below is a table of some commonly used format specifiers:

Format Specifier Description
%c For character type.
%d For signed integer type.
%e or %E For scientific notation of floats.
%f For float type.
%g or %G For float type with the current precision.
%i Signed integer.
%ld or %li For long integer.
%lf For double type.
%Lf For long double type.
%lu For unsigned int or unsigned long.
%lli or %lld For long long integer.
%llu For unsigned long long integer.
%o For octal representation.
%p For pointer.
%s For string.
%u For unsigned int.
%x or %X For hexadecimal representation.
%n Prints nothing.
%% Prints % character.

Understanding the printf() Function in C

The printf() function in C is used to print formatted output to the standard output stdout (usually the console screen). It is part of the C standard library <stdio.h> and allows formatting output in various ways.

Syntax of printf

printf("formatted_string", arguments_list);

Parameters:

  • formatted_string: A string specifying the data to be printed, including format specifiers for variable values.
  • arguments_list: Variables that correspond to the format specifiers in the formatted string.

Return Value: printf() returns the number of characters printed. If an error occurs, a negative value is returned.

Example of printf

In this example, the printf() function is used to print the string "Hello Geek!" to the output.


// C program to illustrate the use of printf function
#include <stdio.h>

int main()
{
// using printf to print "Hello Geek!"
printf("Hello Geek!");

return 0;
}
Output:
Hello Apeksha!

Formatting with printf

The printf() function in C can display various data types using format specifiers that start with the percentage symbol %.

Syntax of Format Specifier

%[flags][width][.precision][length]specifier

Components of Format Specifier

  • Specifier: Denotes the type of data (e.g., %d for integers, %f for floats, %c for characters).
  • Width: Specifies the minimum number of characters to print.
  • Precision: Determines the number of digits after the decimal (for floats) or minimum number of digits (for integers).
  • Length: Modifies the data type (e.g., l for long int).

Examples of Format Specifiers

Specifier Description
%dFor signed integers.
%fFor floating-point numbers.
%cFor characters.
%sFor strings.
%pFor memory addresses.
%xFor hexadecimal values.

Example: Print Variables with Specifiers

This example demonstrates printing integers along with a string using the %d format specifier.


// C program to print variables
#include <stdio.h>

int main()
{
int num1 = 99;
int num2 = 1;
printf("The sum of %d and %d is %d\n", num1, num2, num1 + num2);
return 0;
}
Output:
The sum of 99 and 1 is 100

Understanding the scanf() Function in C

The scanf() function in C stands for "Scan Formatted String" and is used to read data from standard input (stdin, typically the keyboard) and store it in the provided variables.

It accepts character, string, and numeric data from the user using specific format specifiers.

Syntax of scanf

int scanf(const char *format, ...);

Parameters:

  • int - Return type indicating the number of items read and assigned.
  • format - A string containing format specifiers for each variable to store input values.
  • ... - Indicates the function accepts a variable number of arguments.

Return Value: scanf() returns the number of input values successfully assigned, 0 if no values were assigned, or a negative number if an error occurred.

Example Format Specifiers in scanf

Specifier Description
%dAccepts input for integers.
%ldAccepts input for long integers.
%lldAccepts input for long long integers.
%fAccepts input for real numbers (floats).
%cAccepts input for character types.
%sAccepts input for strings.

Example of Using scanf with a Variable

In this example, the scanf function stores an integer input by the user into the variable var.


int var;
scanf("%d", &var);

Why Use &?

The ampersand & symbol is used to get the address of a variable, allowing scanf to store input directly in that memory location.

Example: Using & with scanf


// C program to implement scanf
#include <stdio.h>

int main() 
{
int a, b;

printf("Enter first number: ");
scanf("%d", &a);

printf("Enter second number: ");
scanf("%d", &b);

printf("A : %d \t B : %d", a, b);

return 0;
}
Output:
Enter first number: 5
Enter second number: 6
A : 5 B : 6
