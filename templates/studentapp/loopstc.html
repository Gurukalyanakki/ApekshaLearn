Loops in C Programming

Loops in programming are used to repeat a block of code until the specified condition is met. A loop statement allows programmers to execute a statement or group of statements multiple times without repetition of code.

C Program to Illustrate the Need for Loops

Sample Code

#include <stdio.h>

int main() {
    printf("Hello World\n");
    printf("Hello World\n");
    printf("Hello World\n");
    printf("Hello World\n");
    printf("Hello World\n");
    printf("Hello World\n");
    printf("Hello World\n");
    printf("Hello World\n");
    printf("Hello World\n");
    printf("Hello World\n");

    return 0;
}

Output:

Hello World (10 times)

Types of Loops in C Programming

There are mainly two types of loops in C Programming:

  • Entry Controlled Loops: In Entry controlled loops, the test condition is checked before entering the main body of the loop. For Loop and While Loop are entry-controlled loops.
  • Exit Controlled Loops: In Exit controlled loops, the test condition is evaluated at the end of the loop body, ensuring the loop body executes at least once. The do-while loop is an exit-controlled loop.

Loop Types

  • for loop: First initializes, then checks the condition, then executes the body, and finally performs the update.
  • while loop: First initializes, checks the condition, then executes the body. Updating can be inside the loop body.
  • do-while loop: Executes the body first, then checks the condition.

for Loop

The for loop in C programming is a repetition control structure that allows programmers to execute a block of code a specific number of times.

for (initialize expression; test expression; update expression) {
    // body of for loop
}

Example:

Sample Code

#include <stdio.h>

int main() {
    for (int i = 0; i < 10; ++i) {
        printf("Hello World\n");
    }
    return 0;
}

while Loop

The while loop executes based on a condition and may run indefinitely if the condition remains true.

Sample Code

#include <stdio.h>

int main() {
    int i = 2;
    while (i < 10) {
        printf("Hello World\n");
        i++;
    }
    return 0;
}

Diagram:

do-while Loop

The do-while loop will execute the loop body at least once, regardless of whether the condition is initially true or false.

Sample Code

#include <stdio.h>

int main() {
    int i = 2;
    do {
        printf("Hello World\n");
        i++;
    } while (i < 1);
    return 0;
}

Loop Control Statements

Loop control statements in C are used to modify the normal sequence of loop execution.

  • break statement: Terminates the loop or switch statement, transferring execution to the statement immediately following it.

  • continue statement: Skips the remaining loop body and resets the condition for the next iteration.
