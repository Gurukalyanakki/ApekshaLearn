Operators in C

In C language, operators are symbols that represent operations to be performed on one or more operands.

What is a C Operator?

An operator in C is a symbol that helps perform mathematical, relational, bitwise, conditional, or logical computations on values and variables.

The values and variables used with operators are called operands.

Types of Operators in C

C language provides a wide range of operators classified into six types:

  • Arithmetic Operators
  • Relational Operators
  • Logical Operators
  • Bitwise Operators
  • Assignment Operators
  • Other Operators

1. Arithmetic Operators in C

The arithmetic operators are used to perform arithmetic/mathematical operations on operands. Here is a list of arithmetic operators:

Symbol Operator Description Syntax
+PlusAdds two numeric valuesa + b
-MinusSubtracts right operand from lefta - b
*MultiplyMultiplies two numeric valuesa * b
/DivideDivides two numeric valuesa / b
%ModulusReturns the remaindera % b

Example of Arithmetic Operators in C

Sample Code

#include <stdio.h>
int main() {
  int a = 25, b = 5;
  printf("a + b = %d\\n", a + b);
  printf("a - b = %d\\n", a - b);
  printf("a * b = %d\\n", a * b);
  printf("a / b = %d\\n", a / b);
  printf("a %% b = %d\\n", a % b);
  return 0;
}

Sample output

Output:
a + b = 30
a - b = 20
a * b = 125
a / b = 5
a % b = 0

Unary Operators in C

Unary operators perform operations on a single operand to produce a new value. Here are the types and examples of unary operators in C.

1. Unary Minus (-)

The minus operator changes the sign of its argument. It converts a positive number to negative and vice versa.

Sample Code

#include <stdio.h>
int main() {
  int positiveInteger = 100;
  int negativeInteger = -positiveInteger;
  printf("Positive Integer = %d\\n", positiveInteger);
  printf("Negative Integer = %d", negativeInteger);
  return 0;
}
Output:
Positive Integer = 100
Negative Integer = -100

2. Increment (++)

The increment operator increases the value of a variable by 1. It can be used in two ways:

2.1 Prefix Increment (++a)

The value is incremented before it is used.

2.2 Postfix Increment (a++)

The value is incremented after it is used.

Sample Code

#include <stdio.h>
int main() {
  int a = 5;
  int b = 5;
  printf("Pre-Incrementing a = %d\\n", ++a);
  printf("Post-Incrementing b = %d", b++);
  return 0;
}

3. Decrement (--)

The decrement operator decreases the value of a variable by 1. It can be used in two ways:

3.1 Prefix Decrement (--a)

The value is decremented before it is used.

3.2 Postfix Decrement (a--)

The value is decremented after it is used.

Sample Code

#include <stdio.h>
int main() {
  int a = 5;
  int b = 5;
  printf("Pre-Decrementing a = %d\\n", --a);
  printf("Post-Decrementing b = %d", b--);
  return 0;
}

Bitwise Operators in C

Bitwise operators allow precise manipulation of bits, giving you control over hardware operations. Learn how they work in C and explore advanced usage with practical examples.

Bitwise Operators in C

  • & (bitwise AND): Returns 1 if both bits are 1.
  • | (bitwise OR): Returns 1 if any bit is 1.
  • ^ (bitwise XOR): Returns 1 if bits are different.
  • << (left shift): Shifts bits left by a specified number.
  • > (right shift): Shifts bits right by a specified number.
  • ~ (bitwise NOT): Inverts all bits of a number.

Example of Bitwise Operators in C

Sample Code


// C Program to demonstrate use of bitwise operators

#include <stdio.h>
int main() {
unsigned int a = 5, b = 9;

printf("a = %u, b = %u\n", a, b);
printf("a&b = %u\n", a & b);
printf("a|b = %u\n", a | b);
printf("a^b = %u\n", a ^ b);
printf("~a = %u\n", a = ~a);
printf("b<<1 = %u\n", b << 1);
printf("b>>1 = %u\n", b >> 1);

return 0;
}

Output:


a = 5, b = 9
a&b = 1
a|b = 13
a^b = 12
~a = 4294967290
b<<1 = 18
b>>1 = 4

Interesting Facts About Bitwise Operators

    Shifts on Negative Numbers: Left-shift and right-shift with negative numbers are undefined.

    Bitwise OR as Sum: The bitwise OR of two numbers gives their sum if there's no carry.

    XOR in Problems: Used for finding odd-occurring elements, missing numbers, etc.

    Logical vs. Bitwise: Logical operators return 0 or 1, bitwise operators return integer values.

    Shift Operators as Multiplication/Division: Left-shift/right-shift are equivalent to multiplication/division by 2.

    Odd/Even Check: The & operator can check if a number is odd or even.

Additional Example Programs

Find Odd Occurring Element in an Array

Sample Code


// C program to find odd occurring elements in an array

#include <stdio.h>

int findOdd(int arr[], int n) {
int res = 0, i;
for (i = 0; i < n; i++)
res ^= arr[i];
return res;
}

int main() {
int arr[] = { 12, 12, 14, 90, 14, 14, 14 };
int n = sizeof(arr) / sizeof(arr[0]);
printf("The odd occurring element is %d ", findOdd(arr, n));
return 0;
}

Output: The odd occurring element is 90

Difference between & and && Operators

Sample Code


// C program to demonstrate difference between & and &&

#include <stdio.h>

int main() {
int x = 2, y = 5;
(x & y) ? printf("True ") : printf("False ");
(x && y) ? printf("True ") : printf("False ");
return 0;
}

Output: False True
