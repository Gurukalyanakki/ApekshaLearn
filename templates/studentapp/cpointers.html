Pointers in C Programming Language

Pointers are one of the core components of the C programming language. A pointer can be used to store the memory address of other variables, functions, or even other pointers.

The use of pointers allows low-level memory access, dynamic memory allocation, and many other functionalities in C.

In this article, we will discuss C pointers in detail, their types, uses, advantages, and disadvantages with examples.

What is a Pointer in C?

A pointer is defined as a derived data type that can store the address of other C variables or a memory location. We can access and manipulate the data stored in that memory location using pointers.

Syntax of C Pointers

The syntax of pointers is similar to variable declaration in C, but we use the * (dereferencing operator) in the pointer declaration.

Sample Code

datatype *ptr;

Where:

  • ptr is the name of the pointer.
  • datatype is the type of data it is pointing to.

How to Use Pointers?

The use of pointers in C can be divided into three steps:

  1. Pointer Declaration
  2. Pointer Initialization
  3. Pointer Dereferencing

Pointer Declaration

In pointer declaration, we only declare the pointer but do not initialize it. Example:

Sample Code

int *ptr;

Pointer Initialization

Pointer initialization is the process of assigning an initial value to the pointer. Example:

Sample Code

int var = 10;
    int *ptr;
    ptr = &var;

Pointer Dereferencing

Dereferencing a pointer is the process of accessing the value stored in the memory address specified in the pointer. Example:

Sample Code

printf("Value at *ptr = %d \n", *ptr);

Types of Pointers in C

Pointers in C can be classified into several types, such as:

  • Integer Pointers
  • Array Pointers
  • Structure Pointers
  • Function Pointers
  • Double Pointers
  • Null Pointers
  • Void Pointers
  • Wild Pointers

C Pointer Example

Here’s a sample program illustrating pointer usage in C:

Sample Code

// C program to illustrate Pointers
    #include <stdio.h>
    
    void apeksha() {
        int var = 10;
        int *ptr = &var;
        printf("Value at ptr = %p \n", ptr);
        printf("Value at var = %d \n", var);
        printf("Value at *ptr = %d \n", *ptr);
    }
    
    int main() {
        apeksha();
        return 0;
    }

Advantages of Pointers

  • Dynamic memory allocation and deallocation.
  • Efficient access to array or structure elements.
  • Useful in complex data structures such as linked lists, trees, etc.

Disadvantages of Pointers

  • Pointers are prone to errors, leading to memory corruption.
  • Understanding and using pointers can be complex for beginners.

Double Pointers in C

A pointer to a pointer in C is used when we want to store the address of another pointer. The first pointer is used to store the address of a variable, and the second pointer is used to store the address of the first pointer, hence the term "double pointer."

Declaration of Pointer to a Pointer in C

Declaring a pointer to a pointer in C is similar to declaring a single pointer, except an additional '*' is required:

Sample Code

data_type_of_pointer **name_of_variable = &normal_pointer_variable; int val = 5; int *ptr = &val; // storing address of val to pointer ptr. int **d_ptr = &ptr; // pointer to a pointer

Sample Code

Example of Double Pointer in C

The code below demonstrates how to use a pointer to a pointer to access values:

Sample Code

// C program to demonstrate pointer to pointer #include <stdio.h> int main() { int var = 789; int* ptr2; // pointer to var int** ptr1; // double pointer // storing address of var in ptr2 ptr2 = &var; // storing address of ptr2 in ptr1 ptr1 = &ptr2; // Displaying value of var using both single and double pointers printf("Value of var = %d\n", var); printf("Value of var using single pointer = %d\n", *ptr2); printf("Value of var using double pointer = %d\n", **ptr1); return 0; }

Size of Pointer to Pointer in C

In C, a double pointer typically occupies the same amount of space as a normal pointer. This size can be confirmed with the following code:

Sample Code

// C program to find the size of pointer to pointer #include <stdio.h> int main() { int a = 5; int* ptr = &a; int** d_ptr = &ptr; // size of single pointer printf("Size of normal Pointer: %lu\n", sizeof(ptr)); // size of double pointer printf("Size of Double Pointer: %lu\n", sizeof(d_ptr)); return 0; }

Application of Double Pointers in C

Double pointers have a range of applications in C programming:

  • Used in dynamic memory allocation of multidimensional arrays.
  • Useful in storing multilevel data, such as paragraphs and sentences in a document.
  • Employed in data structures to directly manipulate node addresses without copying.
  • Can serve as function arguments to manipulate addresses stored in local pointers.

Multilevel Pointers in C

Beyond double pointers, C also supports multilevel pointers. For instance, a triple pointer, int ***t_ptr, is a pointer to a double pointer.

Syntax of Triple Pointer

Sample Code

pointer_type ***pointer_name;

Function Pointers in C

In C, similar to normal data pointers (e.g., int *, char *), we can also have pointers to functions. This is a powerful feature in C programming.

Example of Function Pointer Declaration and Call

The following example demonstrates how to declare and use a function pointer:

Sample Code

#include <stdio.h> // A normal function with an int parameter and void return type void fun(int a) { printf("Value of a is %d\n", a); } int main() { // fun_ptr is a pointer to function fun() void (*fun_ptr)(int) = &fun; // Equivalent to: // void (*fun_ptr)(int); // fun_ptr = &fun; // Calling fun() using fun_ptr (*fun_ptr)(10); return 0; }

Explanation of Function Pointer Syntax

The expression void (*fun_ptr)(int) needs extra parentheses to ensure the correct interpretation. Without them, it would be parsed as void *fun_ptr(int), which is a declaration of a function that returns a void pointer.

Interesting Facts about Function Pointers

  1. Unlike normal pointers, a function pointer points to code, not data.
  2. Function pointers do not allocate or de-allocate memory.
  3. A function's name can be used to get its address.

Example Without Address and Dereference Operators

In this example, we remove the address operator & and the dereference operator *:

Sample Code

#include <stdio.h> void fun(int a) { printf("Value of a is %d\n", a); } int main() { void (*fun_ptr)(int) = fun; // & removed fun_ptr(10); // * removed return 0; }

Array of Function Pointers

We can have an array of function pointers. Here’s an example:

Sample Code

#include <stdio.h> void add(int a, int b) { printf("Addition is %d\n", a + b); } void subtract(int a, int b) { printf("Subtraction is %d\n", a - b); } void multiply(int a, int b) { printf("Multiplication is %d\n", a * b); } int main() { void (*fun_ptr_arr[])(int, int) = { add, subtract, multiply }; unsigned int ch, a = 15, b = 10; printf("Enter Choice: 0 for add, 1 for subtract and 2 for multiply\n"); scanf("%d", &ch); if (ch > 2) return 0; (*fun_ptr_arr[ch])(a, b); return 0; }

Function Pointers as Parameters

Function pointers can be passed as arguments. Here’s an example:

Sample Code

#include <stdio.h> void fun1() { printf("Fun1\n"); } void fun2() { printf("Fun2\n"); } void wrapper(void (*fun)()) { fun(); } int main() { wrapper(fun1); wrapper(fun2); return 0; }

Using Function Pointers with qsort

The qsort function uses function pointers to compare elements. Here’s an example:

Sample Code

#include <stdio.h> #include <stdlib.h> int compare(const void* a, const void* b) { return (*(int*)a - *(int*)b); } int main() { int arr[] = { 10, 5, 15, 12, 90, 80 }; int n = sizeof(arr) / sizeof(arr[0]), i; qsort(arr, n, sizeof(int), compare); for (i = 0; i < n; i++) printf("%d ", arr[i]); return 0; }

General Purpose Search Function

Here is a general-purpose search function that can be used with any data type using function pointers:

Sample Code

#include <stdbool.h> #include <stdio.h> bool compare(const void* a, const void* b) { return (*(int*)a == *(int*)b); } int search(void* arr, int arr_size, int ele_size, void* x, bool compare(const void*, const void*)) { char* ptr = (char*)arr; int i; for (i = 0; i < arr_size; i++) if (compare(ptr + i * ele_size, x)) return i; return -1; } int main() { int arr[] = { 2, 5, 7, 90, 70 }; int n = sizeof(arr) / sizeof(arr[0]); int x = 7; printf("Returned index is %d ", search(arr, n, sizeof(int), &x, compare)); return 0; }

NULL Pointers in C

A NULL pointer is a pointer that does not point to any memory location but has the value NULL. According to the C11 standard:

“An integer constant expression with the value 0, or such an expression cast to type void *, is called a null pointer constant. If a null pointer constant is converted to a pointer type, the resulting pointer, called a null pointer, is guaranteed to compare unequal to a pointer to any object or function.”

Syntax of NULL Pointer Declaration in C

We can declare a NULL pointer as follows:

Sample Code

type pointer_name = NULL; type pointer_name = 0;

NULL expands to an implementation-defined null pointer constant. It is defined in header files such as stdio.h, stddef.h, and stdlib.h.

Uses of NULL Pointer in C

  • To initialize a pointer variable when it hasn’t been assigned any valid memory address yet.
  • To check for a null pointer before accessing any pointer variable, allowing error handling in pointer-related code.
  • To pass a null pointer to a function argument when we don’t want to pass any valid memory address.
  • NULL pointers are used in data structures like trees and linked lists to indicate the end.

Checking if a Pointer is NULL

To check if a pointer is NULL, we use the == operator:

Sample Code

ptr == NULL;

Examples of NULL Pointers in C

Example 1: Avoiding Segmentation Fault with NULL Pointer

This program checks if a pointer is NULL before dereferencing it:

Sample Code

#include <stdio.h> int main() { int* ptr = NULL; if (ptr == NULL) { printf("Pointer does not point to anything"); } else { printf("Value pointed by pointer: %d", *ptr); } return 0; }

Example 2: Checking Memory Allocation with malloc()

The malloc() function returns NULL if memory allocation fails. We can check this to ensure successful allocation:

Sample Code

#include <stdio.h> #include <stdlib.h> int main() { int* ptr = (int*)malloc(5 * sizeof(int)); if (!ptr) { printf("Memory Allocation Failed"); exit(0); } return 0; }

Example 3: Dereferencing a NULL Pointer

If we try to dereference a NULL pointer, it may cause a crash depending on the machine:

Sample Code

#include <stdio.h> int main() { int* ptr = NULL; printf("%d", *ptr); // Dereferencing NULL pointer return 0; }

Example 4: Passing NULL to a Function

We can pass a NULL pointer to a function as shown here:

Sample Code

#include <stdio.h> void foo(int* ptr) { if (ptr == NULL) { printf("NULL Pointer Passed"); return; } printf("Non-Null Pointer Passed"); } int main() { foo(NULL); return 0; }

Difference Between NULL Pointer and Void Pointer in C

NULL Pointer Void Pointer
A NULL pointer does not point to anything. It is a special reserved value for pointers. A void pointer points to a memory location that may contain typeless data.
Any pointer type can be assigned NULL. It can only be of type void.
All NULL pointers are equal. Void pointers can be different.
NULL Pointer is a value. A void pointer is a type.
Example: int *ptr = NULL; Example: void *ptr;
