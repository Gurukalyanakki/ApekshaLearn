Variables in C

What is a Variable in C?

A variable in C is a memory location with a specific name that allows the storage and retrieval of data. Different data types can be stored, and the same variable can be used repeatedly to store new data.

C Variable Syntax


int variable_name;       // Single variable declaration
data_type var1, var2;    // Multiple variable declarations
Component Description
data_type Specifies the type of data the variable will store.
variable_name The name of the variable defined by the user.
value An optional initial value assigned to the variable.

Examples of Variable Declarations


int var;    // integer variable
char a;     // character variable
float fff;  // float variable

Steps for Defining a Variable

Step Description

Example Code


#include <stdio.h>

int main() {
    int var = 10;     // Variable initialization
    printf("Value of var: %d\n", var);
    return 0;
}

Rules for Naming Variables in C

Rule Explanation
Use only letters, digits, and underscores Variable names can contain alphabets, digits, and underscores.
Start with a letter or underscore Names cannot begin with a digit.
No spaces Variable names must not contain spaces.
Avoid keywords Variable names cannot be reserved words.

Variable Types in C

Variable Type Description
Local Variables Declared inside a function, accessible only within that function.
Global Variables Declared outside all functions, accessible from any function in the program.
Static Variables Retain their value between function calls.
Automatic Variables Local variables with automatic storage duration (default).
Extern Variables Declared with the extern keyword, usually defined in another file.
Register Variables Stored in a CPU register for fast access, declared with register keyword.

Understanding Constants in C Programming

In C programming, constants are fixed values that cannot be altered during the program execution. They are essential for values that should remain unchanged, like mathematical constants and limits.

Types of Constants

  • Integer Constants: Whole numbers without a decimal point.
  • Floating-point Constants: Numbers with a decimal or in exponential notation.
  • Character Constants: Single characters enclosed in single quotes.
  • String Constants: Sequences of characters in double quotes.
  • Enumeration Constants: User-defined constants in an enumeration type.

Example Code

Below is an example demonstrating the use of constants in C:

#include <stdio.h> #define PI 3.14159 // Macro constant for Pi int main() { const int MAX_USERS = 100; // Constant variable for max users printf("PI: %f\\n", PI); printf("Max Users: %d\\n", MAX_USERS); return 0; }

Expected Output

PI: 3.141590
Max Users: 100

Explanation

In this code:

  • #define PI 3.14159 is a macro constant, which the preprocessor replaces wherever "PI" is found.
  • const int MAX_USERS = 100; declares a constant variable that cannot be changed after initialization.
