C Programming Language Tutorial

What is C?

C is a general-purpose, procedural, high-level programming language used in the development of software, system programming, games, and more.

  • C was developed by Dennis M. Ritchie at Bell Telephone Laboratories in 1972.
  • It is powerful, flexible, and initially created for programming UNIX.
  • Today, C is one of the most widely used languages worldwide.

C is known for its simplicity and efficiency, making it a great choice to start learning programming.

Sample Code


 #include <stdio.h>
 
 int main() {
     printf("Hello, World!");
     return 0;
 }

Key Features of C Language

  • Procedural Language: Follows a structured approach with functions and procedures.
  • Fast and Efficient: Great memory management and quick execution.
  • Modularity: Organizes code into functions or modules.
  • Statically Typed: Variables require explicit data type definitions.
  • General-Purpose: Suitable for diverse applications, from system to application development.
  • Rich Set of Operators: Supports a comprehensive set of operations.
  • Libraries with Rich Functions: Offers a broad library of functions.
  • Middle-Level Language: Bridges high- and low-level programming.
  • Portability: C programs can run on different platforms with minor adjustments.
  • Easy to Extend: Allows new features or functions to be added easily.

Comments in C Programming

Comments in C are notes in the source code, helping to make the code understandable without execution by the compiler or interpreter.

When and Why to Use Comments?

  • Helpful for large codebases.
  • Describes algorithms for clarity.
  • Allows deactivation of code temporarily.

Types of Comments in C

  • Single-line comment
  • Multi-line comment

1. Single-line Comment

A single-line comment starts with // and extends to the end of the line.

Syntax:

// This is a single-line comment

Example: Single-line comment in a C program


 // C program to illustrate single-line comment
 #include <stdio.h>
 
 int main(void) {
     // This is a single-line comment
     printf("Welcome to GeeksforGeeks");
     return 0;
 }

2. Multi-line Comment

A multi-line comment begins with /* and ends with */.

Syntax:

/*
 Multi-line comment starts here
 continues here
 ends here
 */
